## How to Add Affiliate Links

### Option 1: Text Links in Articles
```html
<a href="https://partner-link.com" rel="sponsored" target="_blank">
  Try Claude Code
</a>
```

### Option 2: Call-to-Action Boxes
Add to your articles:
```html
<div class="affiliate-box">
  <h4>Try This AI Tool</h4>
  <p>We recommend Claude Code for developers.</p>
  <a href="affiliate-link" class="cta-button">Get Started</a>
  <small>* Affiliate link - supports our work</small>
</div>
```

### Option 3: Resource Pages
Create dedicated pages:
- "Best AI Coding Tools"
- "Top LLM APIs"
- "AI Infrastructure Providers"
