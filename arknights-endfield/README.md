# Arknights: Endfield Guide

A comprehensive guide website for Arknights: Endfield featuring operators, tier lists, builds, and team compositions.

## 📁 Structure

```
arknights-endfield/
├── index.html              # Main landing page with tier list, operators grid
├── characters/
│   └── character.html      # Dynamic character detail page
├── css/
│   ├── style.css          # Main stylesheet
│   └── character.css      # Character page styles
├── js/
│   └── character-data.js  # All operator data
└── images/                # Local image assets
```

## 🎮 Features

- **Operator Tier List** - S, A, B, C tier rankings
- **Character Database** - 12+ operators with detailed stats
- **Build Guides** - Weapons, modules, and team compositions
- **Element System** - Elemental reactions and inflictions
- **Responsive Design** - Mobile-friendly layout

## 🔗 Data Sources

- Operator data: endfield.wiki.gg
- Images: endfield.wiki.gg, official Hypergryph assets
- Stats and abilities: In-game data

## 🚀 Deployment

This site is designed for GitHub Pages deployment. Simply push to the repository and enable GitHub Pages.
