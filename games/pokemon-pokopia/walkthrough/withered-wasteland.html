Overview

Withered Wasteland is the starting area of Pokemon Pokopia. When you first arrive, you'll meet Professor Tangrowth who will guide you through the basics of the game.

This area features dry, desert-like terrain with scattered vegetation and abandoned structures.

Pokemon Found

The following Pokemon can be found in Withered Wasteland:

  • Bulbasaur - Grass/Poison type
  • Squirtle - Water type
  • Charmander - Fire type
  • Caterpie - Bug type
  • Weedle - Bug/Poison type
  • Pidgey - Normal/Flying type
  • Rattata - Normal type

Items

Various items can be found throughout Withered Wasteland:

  • Oran Berries
  • Pecha Berries
  • Leppa Berries
  • Sturdy Sticks
  • Small Logs

Requests

Complete these requests in Withered Wasteland:

  • Water Dry Grass
  • Help Me Make a Home
  • Build a Leaf Den

Tips

  • Focus on building habitats for Grass-type Pokemon first
  • Use Water Gun to revive dry plants
  • Complete all requests to unlock new areas
  • Talk to Professor Tangrowth for guidance
