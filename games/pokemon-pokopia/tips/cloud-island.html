Cloud Island Guide

Visit other players' islands online

Home › Tips › Cloud Island

Cloud Islands allow you to visit other players' islands online and get exclusive items.

How to Visit

  1. Unlock Cloud Island access (story progress)
  2. Go to the pier
  3. Select "Visit Cloud Island"
  4. Choose an island to visit

What to Do

  • Collect items left by players
  • Complete exclusive requests
  • Get rare items to sell
  • Share your island code
