Time Travel Guide

How to change the in-game time and date

Home › Tips › Time Travel

Learn how to time travel in Pokemon Pokopia to change the in-game date and access different events or daily content.

How to Time Travel

  1. Go to the Nintendo Switch Settings
  2. Select "System"
  3. Choose "Date and Time"
  4. Turn off "Synchronize Clock via Internet"
  5. Manually change the date and time
  6. Launch Pokemon Pokopia
Warning: Time traveling may affect certain game mechanics. Some events and daily activities reset based on the game clock.

What Changes with Time Travel

  • Daily quests refresh
  • Berry growth cycles
  • Event availability
  • Wild Pokemon spawns

Tips

  • Forward time travel can be done freely
  • Backward time travel may reset some daily activities
  • Use time travel to grind daily rewards
