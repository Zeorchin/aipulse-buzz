Controls Guide

Controller and keyboard layouts for Pokemon Pokopia

Home › Tips › Controls

Controller Controls

Movement

  • Left Stick: Move character
  • D-Pad: Move character

Actions

  • A Button: Interact / Confirm
  • B Button: Cancel / Run
  • X Button: Open menu
  • Y Button: Use active move
  • ZR Button: Use active move

Camera

  • Right Stick: Camera control
  • L / R: Change active move

Keyboard Controls

  • WASD: Move
  • E: Interact
  • Q: Use move
  • Space: Jump
  • Esc: Menu
